# Open Claude Desktop

Proxy URL / API Base URL을 GUI로 설정할 수 있는 오픈소스 Claude 데스크톱 클라이언트.

## 기술 스택

TypeScript, Electron, React, Claude Agent SDK

## 마음가짐

### 단순함을 추구한다

복잡한 해결책보다 단순한 해결책을 선택한다. 지금 필요하지 않은 것은 만들지 않는다. 복잡함이 불가피할 때는 SOLID 원칙을 따른다.

### 읽기 쉬운 코드를 작성한다

코드는 작성하는 시간보다 읽는 시간이 더 길다. 미래의 독자를 배려한다.

### 테스트가 설계를 이끈다

테스트를 먼저 작성한다. 테스트하기 어렵다면 설계를 다시 생각한다.
