# 기능 목록

## FR-1: A2UI (Agent-to-UI)

Agent 응답을 실시간으로 UI에 렌더링한다.

| ID | 기능 | 설명 |
|----|------|------|
| FR-1.1 | 텍스트 스트리밍 | 응답 텍스트를 청크 단위로 표시 |
| FR-1.2 | 마크다운 렌더링 | 제목, 목록, 링크, 표 등 지원 |
| FR-1.3 | 코드 블록 | 문법 하이라이팅, 복사 버튼 |
| FR-1.4 | 도구 사용 표시 | 도구 호출과 결과를 시각화 |
| FR-1.5 | 인터랙티브 컴포넌트 | 버튼, 폼, 차트 등 |

---

## FR-2: Workspace

프로젝트 컨텍스트를 관리한다.

| ID | 기능 | 설명 |
|----|------|------|
| FR-2.1 | 작업 디렉토리 설정 | 대화의 기준 경로 지정 |
| FR-2.2 | 프로젝트 설정 | `.claude/` 디렉토리 기반 설정 격리 |
| FR-2.3 | 디렉토리 전환 | 다른 프로젝트로 컨텍스트 전환 |

---

## FR-3: 인증

Claude API 인증을 관리한다.

| ID | 기능 | 설명 |
|----|------|------|
| FR-3.1 | OAuth 로그인 | `claude login` CLI 위임 |
| FR-3.2 | API Key 입력 | 직접 API Key 입력 (대안) |
| FR-3.3 | 인증 상태 표시 | 로그인 여부와 이메일 표시 |
| FR-3.4 | 토큰 자동 참조 | `~/.claude/` 인증 정보 사용 |

---

## FR-4: 설정

API 연결 설정을 관리한다.

| ID | 기능 | 설명 |
|----|------|------|
| FR-4.1 | Base URL 설정 | 커스텀 API 엔드포인트 지정 |
| FR-4.2 | Proxy URL 설정 | HTTP/HTTPS 프록시 지정 |
| FR-4.3 | 연결 테스트 | 설정 유효성 검증 |
| FR-4.4 | 설정 저장 | 앱 재시작 후에도 유지 |
| FR-4.5 | 설정 위계 | ~/.claude → .claude 병합 |

---

## FR-5: 확장 설정 (Future)

Plugin, MCP 등 확장 기능을 GUI로 설정한다.

| ID | 기능 | 설명 |

## MVP 범위

### 포함

- FR-1.1 ~ FR-1.4 (기본 A2UI)
- FR-2.1 (작업 디렉토리)
- FR-3.1, FR-3.3, FR-3.4 (OAuth 인증)
- FR-4.1 ~ FR-4.4 (기본 설정)

### 제외 (MVP 이후)

- FR-1.5 (인터랙티브 컴포넌트)
- FR-2.2, FR-2.3 (고급 워크스페이스)
- FR-3.2 (API Key 직접 입력)
- FR-5.x (확장 설정)
